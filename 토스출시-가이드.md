## 사용자님이 직접 해야 하는 것 (본인 인증 필요)

### 1. 콘솔 가입 + 앱 등록 (약 5분)
1. https://developers-apps-in-toss.toss.im/ → **콘솔** 접속
2. 본인 명의 토스앱으로 가입 (만 19세 이상)
3. 앱 등록 — 3가지만 입력:
   - **앱 이름**: `연기공간` (나중에 수정 가능)
   - **appName**: `smoke-finder` (고유 ID, 수정 불가 — 다른 걸 쓰면 저에게 알려주세요)
   - **앱 유형**: `비게임`

### 2. NCP 도메인 추가 (앱 등록 후)
토스가 미니앱을 자체 CDN에 호스팅하므로, 콘솔에서 안내되는 도메인을
NCP 콘솔 → Maps Application → **Web 서비스 URL**에 추가해야 네이버 지도가 뜹니다.
(도메인은 콘솔 등록 후 확인 가능 — 그때 제가 안내)

### 3. (선택) 사업자 등록
사업자 없이도 출시 가능. 광고·인앱결제 등 수익화하려면 사업자 등록 후 콘솔에서 서류 제출 (검토 1~2 영업일).

## 심사용 앱 소개 문구 (공익 포지셔닝)

> **한 줄 소개**: 지정 흡연구역만 골라 안내하는 위치 기반 지도 서비스
>
> **상세 소개**:
> 연기공간은 흡연자가 **지정된 흡연구역에서만** 흡연하도록 돕는 위치 기반 안내 서비스입니다.
> 전국 17개 지자체가 공공데이터포털에 개방한 공식 흡연구역 데이터와 OpenStreetMap 데이터를 기반으로,
> 현재 위치에서 가장 가까운 지정 흡연구역과 도보 경로를 안내합니다.
>
> 무분별한 길거리 흡연과 간접흡연 피해를 줄이고, 금연구역 흡연으로 인한 과태료를 예방하는 것이
> 서비스의 목적입니다. 담배 판매·광고·구매 유도 기능은 일절 없습니다.
>
> - 회원가입 없음, 위치정보 서버 저장 없음 (기기 내에서만 사용)
> - 데이터 출처: 공공데이터포털(지자체 개방 데이터, 공공누리), OpenStreetMap

## 심사 대비 메모

- **4단계 검수**: 운영(앱 정보·서류) → 기능(오류) → 디자인(토스 UI 가이드) → 보안(개인정보)
- **위치 권한**: granite.config.ts에 `geolocation` 권한 선언 필수. 처리방침에 "기기 내 사용, 서버 전송 없음" 명시됨 ✓
- **디자인 검수 리스크**: 커스텀 다크 UI가 지적될 수 있음 → 지적 시 TDS(토스 디자인 시스템) 스타일로 조정
- **콘텐츠 리스크**: 담배 관련 서비스로 분류될 가능성 → 위 공익 포지셔닝 문구 사용, 만 19세 이상 대상 명시
- **반려되어도** 사유가 명시되어 오므로 수정 후 재제출 가능

## 기술 구조 (참고)

```
smoking-area-finder/
├── index.html          ← 본체 (웹/토스 공용, 위치 API 자동 분기)
├── privacy.html        ← 개인정보 처리방침
├── korea-spots.json    ← 공공데이터 293곳
└── toss-app/           ← 토스 미니앱 래퍼 (예정)
    ├── granite.config.ts   (appName, 권한, 브랜드)
    ├── sync.js             (본체 파일 복사 + 어댑터 주입)
    ├── src/toss-adapter.js (SDK 위치 API → window.TossLocation)
    └── dist/               (빌드 산출물 → 콘솔 업로드)
```

배포 흐름: 본체 수정 → git push (웹 반영) + `toss-app`에서 빌드 → 콘솔 업로드 (토스 반영)
